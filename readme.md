# Three CSS Loaders

This repo contains three different CSS loaders for use on web pages.

## Description

1. Gradient Loader
   - Circular design
   - Rotating gradient
   - Glow effect

2. Pulsating Loader
   - Simple circle
   - Pulsation effect

3. Linear Loader
   - Horizontal line
   - Gradient fill
   - Pulsation effect

## Compatibility

The loaders work in modern web browsers with CSS3 support.
